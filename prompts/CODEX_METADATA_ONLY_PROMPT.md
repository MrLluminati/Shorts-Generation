# Codex Metadata Only Prompt

Generate YouTube upload packages for finished MrLluminati Gaming Shorts.

Read:

- `docs/SHORTS_STRATEGY.md`
- `docs/CONTENT_PILLARS.md`
- `docs/TITLE_AND_HOOK_LIBRARY.md`
- `docs/METADATA_TEMPLATE.md`

For each finished Short, create:

- title
- 5 alternative titles
- full description
- short description
- comma-separated tags
- 6-8 targeted hashtags
- 4-5 recommended description hashtags
- 5 thumbnail/on-screen hook text options
- pinned comment

Rules:

- Titles must be accurate to the footage.
- Use viewer-emotion titles over technical labels.
- Do not call Minecraft clips "modded Minecraft" unless the footage is clearly modded.
- Hashtags must match the game and clip type.
- Keep the package easy to copy into YouTube Studio.
