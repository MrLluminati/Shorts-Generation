# YouTube Live Video Editing Pipeline

Use this folder for future videos from your own channel.

## Input

Place downloaded YouTube live recordings in `live_video_pipeline/input/`.

For each live video, include one short note with:

- YouTube URL
- Topic/title
- Any must-keep moments
- Any parts to avoid
- Target language

## Output Types

### Short Form

- 15-60 seconds
- Vertical 1080x1920
- Strong hook in first 1-2 seconds
- Fast captions
- Clean zooms, punch-ins, sound polish, and light effects
- Metadata: title, description, hashtags, tags, pinned comment

### Long Form

- Usually 6-18 minutes, depending on the source
- Horizontal 1920x1080
- Remove dead air, loading, repeated points, long pauses, and off-topic drift
- Keep narrative continuity
- Add intro title, section cards when useful, clean audio, and optional chapter list
- Metadata: title, description, chapters, hashtags, tags, pinned comment

## Folder Map

- `input/` - raw live recordings and source notes
- `work/` - transcripts, markers, intermediate cuts
- `short_form/` - finished Shorts
- `long_form/` - finished long-form edits
- `metadata/` - upload-ready metadata sheets
- `review/` - preview frames and contact sheets

## Standard Deliverable

For each source live video, create:

- 3-8 Shorts
- 1 long-form edited upload
- Metadata CSV and Markdown
- Review frames/contact sheets
- A recommended upload order
