# Upload Checklist

Use this before uploading every Short.

## Clip

- [ ] Is the first second interesting?
- [ ] Does hook text appear immediately?
- [ ] Is the Short 10-18 seconds where possible?
- [ ] If it is 19-22 seconds, is the extra time justified?
- [ ] Is there any dead time at the start?
- [ ] Does the video end immediately after the payoff?
- [ ] Is the action visible on mobile?
- [ ] Is text readable on mobile?
- [ ] Is text kept away from crosshair, enemy, kill feed, car, or main action?
- [ ] Is the audio usable?

## Metadata

- [ ] Does the title create curiosity?
- [ ] Is the title accurate?
- [ ] Are hashtags targeted to the actual video?
- [ ] Are tags comma-separated and relevant?
- [ ] Is the pinned comment easy to answer?
- [ ] Is the clip posted under the correct content pillar?

## Analytics After Upload

- [ ] Wait before judging a paused Short.
- [ ] Do not use repeated self-views as proof of reach.
- [ ] Check shown in feed.
- [ ] Check viewed vs swiped away.
- [ ] Check average view duration.
- [ ] Check average percentage viewed.
- [ ] Log what worked or failed.
