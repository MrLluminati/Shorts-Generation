# Metadata Template

Every finished Short should have a metadata package in this exact order.

```text
TITLE

ALTERNATIVE TITLES
1.
2.
3.
4.
5.

DESCRIPTION

SHORT DESCRIPTION

TAGS

TARGETED HASHTAGS

RECOMMENDED DESCRIPTION HASHTAGS

THUMBNAIL / ON-SCREEN HOOK TEXT
1.
2.
3.
4.
5.

PINNED COMMENT
```

## Title Rules

- Use curiosity, conflict, panic, or payoff.
- Keep it accurate to the footage.
- Put the game name in the title when useful.
- Avoid purely technical titles unless the clip is clearly for a technical audience.

## Description Rules

The first line should explain the clip in plain language. Do not overstuff keywords. Include a light channel/context line only after the main description.

## Tags Rules

Tags should be comma-separated and include:

- game name
- common game variation
- clip type
- channel name
- gameplay category

Example:

```text
cs2, counter strike 2, cs2 shorts, toxic teammates, solo queue, fps gaming, gaming shorts, mrlluminati gaming
```

## Hashtag Rules

Use targeted hashtags that match the actual clip. Do not use unrelated viral tags.
